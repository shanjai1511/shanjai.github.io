Enhanced Chemistry Lab Learning Management System - ECLLMS

The Enhanced Chemistry Lab Learning Management System (ECLLMS) is a cutting-edge educational tool created to completely transform the way chemistry labs teach and learn. It gives students access to a carefully selected collection of chemistry experiments along with comprehensive guidelines and multimedia materials. To accommodate different learning methods, the platform provides interactive features like voice command navigation and AI-powered doubt clearing. By providing tools for experiment administration, customization, and safety protocols, ECLLMS enables teachers to provide a student-centered approach to chemistry instruction.

Tech stack: HTML, CSS, PHP, JavaScript, Bootstrap, MySQL, Python, Langchain, OpenAi, Hugging Face, Manually Collected and Processsed Datasets

Chemistry Lab Experiment Collabrative Teaching System - CLECTS

CLECTS is a pioneering web-based platform that reimagines the teaching and learning of chemistry lab experiments. With a keen focus on accessibility, interactivity, and safety, CLECTS empowers both students and educators. Its student module offers a curated repository of experiments, complete with step-by-step instructions and auditory learning options, making complex concepts accessible to all learning styles. For educators, CLECTS simplifies experiment management, seamlessly integrating safety guidelines and resources. The inclusion of voice command navigation further enhances convenience. Built on a robust tech stack, CLECTS is scalable, ensuring it reaches learners across diverse settings. This project represents an intersection of pedagogy and technology, revolutionizing chemistry education and making it more interactive, accessible, and effective.

Tech stack: HTML, CSS, PHP, JavaScript, Bootstrap, MySQL

Multiple Restaurant Table Booking System

A multiple restaurant table booking system is a comprehensive software solution designed to streamline the process of reserving tables at various dining establishments. This project aims to create a user-friendly platform that enables customers to browse through a list of participating restaurants, view their availability, and make reservations with ease. The system will incorporate features such as user registration and profiles, real-time restaurant availability updates, dynamic menus, and secure payment processing. Additionally, restaurant owners and administrators will have access to a dashboard to manage reservations, update menus, and track customer feedback. The system will also integrate notification mechanisms to confirm reservations and send reminders to customers. By implementing this project, it will enhance the dining experience for customers, optimize restaurant operations, and facilitate efficient communication between diners and eateries, ultimately fostering a thriving ecosystem for both parties.

Tech stack: HTML, CSS, PHP, JavaScript, MySQL

BookMyTab

"BookMyTab" is a console-based restaurant table booking system implemented as a Java project with MySQL integration, serving as an educational endeavor to apply Object-Oriented Programming (OOP) concepts. This project empowers users to interact with a text-based interface where they can explore a list of participating restaurants, check table availability in real-time, and make reservations seamlessly. Users can create profiles, providing a personalized experience, and securely make payments for their bookings. On the administrative side, restaurant owners and managers can efficiently manage reservations, update menus, and gather customer feedback through a command-line dashboard. The system also incorporates notifications to confirm reservations and send timely reminders. "BookMyTab" is a valuable exercise in utilizing OOP principles, such as encapsulation, inheritance, and polymorphism, to model real-world entities like users, restaurants, and reservations, thereby enhancing your understanding of Java and database interactions while simulating a practical use case.

Tech stack: Java, Data Strucutures, OOPS, MySQL
