Modular Web Scraping Pipeline

PythonMySQLBeautifulSoup

A production-scale, multi-site scraping system with YAML-driven site configuration, retry logic, and MySQL-backed storage.

Problem

Site-specific scripts broke constantly with no unified architecture for scaling across multiple data sources.

Approach

Designed a YAML-configured adaptor system separating URL collection, fetching, parsing, and storage as independent stages.

Architecture
URL Collector→ Page Fetcher→ YAML Parser→ BS4 / Scrapy→ Retry Layer→ MySQL
